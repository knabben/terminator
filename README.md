RUN
---

```
cd terminator;
$ npm install
$ node server.js    # HMR and Webpack watchers

python manage.py runserver
```

The bundle assets are hosted on http://localhost:3000, Django runserver access it via template tags.


HELM PORT FORWARDING
---

The Python client (default generated by Swagger codegen) have a fault on it (https://github.com/kubernetes-incubator/client-python/issues/58), it misses a websocket/SDPY, so you must enable the access via port-forward directly or run a local copy of tiller.
